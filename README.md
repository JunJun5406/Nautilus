# Nautilus
<p align="center"><img src="https://github.com/hex3gc/Nautilus/blob/main/Images/expansion.png?raw=true" width=256></p>
<p align="center">Adds 8 void counterparts to vanilla and DLC items.</p>

| Icon  | Description |
| ------------- | ------------- |
| | <p align="center">**VOID COMMON**</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/voidWatch.png?raw=true" width=128> | **Collector's Appraisal**<br>```Deal bonus damage at high health.```<hr><p align="right">Deal **10%** *(+10% per stack)* bonus damage while your health is above 90%. Additional temporary barrier multiplies this bonus up to 2x. ***Corrupts all Delicate Watches.***</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/crabsinthe.png?raw=true" width=128> | **Crabsinthe**<br>```Increase passive health regeneration.```<hr><p align="right">Increases health regeneration by **25%** *(+25% per stack)*. ***Corrupts all Power Elixirs and Empty Bottles.***</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/wellies.png?raw=true" width=128> | **Waterlogged Wellies**<br>```On hit, weaken airborne enemies and pull them to the ground.```<hr><p align="right">Hitting an airborne enemy pulls them downward *(with additional strength per stack)* and **Weakens** them for **2** seconds *(+2 seconds per stack)*. ***Corrupts all Hiker's Boots.***</p> |
| | <p align="center">**VOID UNCOMMON**</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/viscousPot.png?raw=true" width=128> | **Viscous Pot**<br>```Reduce barrier decay, and gain barrier on secondary skill use.```<hr><p align="right">**Reduces barrier decay by 20%.** Using your secondary skill grants **7.5%** *(+7.5% per stack)* temporary barrier. ***Corrupts all Luminous Shots.***</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/paleStar.png?raw=true" width=128> | **Pale Star**<br>```Convert a chest drop into a Void Potential, and recharge with a Void Cradle.```<hr><p align="right">Opening a chest consumes all stacks of this item and converts the chest's reward into a **Void Potential** of equivalent rarity, offering a choice between **3** *(+1 per stack)* **void items**. **All stacks of this item regenerate after a Void Cradle is used.** ***Corrupts all Sale Stars.***</p> |
| | <p align="center">**VOID LEGENDARY**</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/shimmeringNautilus.png?raw=true" width=128> | **Shimmering Nautilus**<br>```Resist and retaliate against repeated strikes.```<hr><p align="right">**Reduce all incoming damage by 10%.** After hitting you 5 times, your attacker evokes an orb that strikes them for **1800% damage** *(+1800% per stack)*. ***Corrupts all Shattering Justices.***</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/motherOfPearl.png?raw=true" width=128> | **Mother Of Pearl**<br>```Collecting a gold chunk fires a volley of missiles.```<hr><p align="right">Collecting a chunk of gold launches **3 missiles** dealing **300%** damage *(+300% per stack)* each. Has a flat 10% chance to convert any pickup into a chunk of gold. ***Corrupts all War Bonds.***</p> |
| <img src="https://github.com/hex3gc/Nautilus/blob/main/Images/xenobacteria.png?raw=true" width=128> | **Xenobacteria**<br>```Increase your entire team's attack speed.```<hr><p align="right">Increase **yours and all your allies' attack speed by 30%** *(+30% per stack)*. ***Corrupts all Alien Heads.***</p> |
